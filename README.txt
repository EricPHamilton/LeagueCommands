Current commands: 
	1. None. Still implementing base.
	
Planned features: 
	1. help
		1. Lists all commands and usage
	2. help <command>
		1. Lists usage for specific command
	3. getchamp <name>
		1. Will open League wiki page associated with <name> champion.
	4. getstat <champ name> <stat>
		1. Will display given stat (armor/l, base armor, base AD, etc.) to output.
	5. opgg <Username> <Region>
		1. Opens the op.gg webpage associated with given data. Region defaults to NA
	6. opgg currentgame <Username> <Region>
		1. Opens up op.gg's live game page.
	7. lolking <Username> <Region>
		1. Same as command opgg, but returns lolking page.
	8. abilitystats <champion> <button>
		1. Returns useful information based on champ spell (Cooldowns at certain ranks, scalings, mana cost
	9. summspellstats <spell>
		1. Returns base CD and when applicable, damage dealt/healed/sheilded.
	10. matchhistory <Username> <Region>
		1. Returns stats based on users match history (common champ, gamemode, etc.)
	11. getchampid <name>
		1. Returns the LoL ID of a given champion
	12. getrank <Summoner> <Region>
		1. Returns solo queue rank of summoner. Region defaults to NA
	13. clear
		1. Clears text in display.
	14. summid <Username> <Region>
		1. Returns the summoner's ID.
	15. update
		1. Checks for program updates 
	16. goldeff <ItemName>
		1. Calculates gold efficiency of a certain item
	17. Ideas are very welcome!
